## 商品管理システム

### 概要

このシステムでは、店舗で扱う商品の在庫管理を行うことができます。

商品の新規登録から編集、削除、画像添付を行うことができます。

### 主な機能
 
 - ログイン・ログアウト機能
 - 商品画面一覧
 - 商品新規登録、編集、削除、画像添付機能

 ### 開発環境

PHP 8.2.12
Laravel 10.13.5
MySQL 15.1
